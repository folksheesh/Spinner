Panduan Penggunaan

1. Kontrol Undian

  • Start / Stop: Tekan Spacebar atau klik area tengah layar.
  • Continue: Saat pemenang muncul, tekan Spacebar / klik layar untuk lanjut.

2. Mode Presentasi

  • Tekan tombol F di keyboard untuk menyembunyikan kursor dan tombol-tombol antarmuka (UI).
  • Tekan F lagi untuk menampilkannya kembali.

3. Remote Control (HP)

  • Buka remote.html di HP Anda.
  • Masukkan Remote Code yang tampil di sudut kiri atas layar utama.
  • Gunakan HP Anda sebagai remote nirkabel untuk mengontrol undian.

4. Ekspor Data

  • Klik tombol 🏆 Summary lalu klik Export CSV untuk mengunduh daftar pemenang.
